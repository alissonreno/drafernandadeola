## 🎯 Sobre o Projeto

Site desenvolvido para fortalecer a presença digital da **Dra. Fernanda Deola**, biomédica esteta (CRBM 010652) localizada em Itajaí/SC. 

O projeto tem como objetivos:
- ✨ Gerar agendamentos via WhatsApp e Instagram
- 💎 Fortalecer a autoridade profissional da Dra. Fernanda
- 🤍 Transformar visitantes em clientes através de uma experiência elegante
- 📱 Funcionar como extensão da marca no Instagram
